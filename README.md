# vue-err
这里是我在写vue的时候遇到的一些问题，记录，总结
# vue-err1
父组件 ：里面有子组件的标签，通过 冒号 绑定本页的数据到 子组件上面去。
子组件 ：可以直接用，然后要在props里面声明一下类型。
子组件怎么传递数据：在方法里面写一个$emit（‘监听的东西’，要传的数据）方法。
父组件需要 在一个标签上 绑定 v-on：方法名=“监听的东西”，在当前页面写一下方法（需要传参数进去）。
